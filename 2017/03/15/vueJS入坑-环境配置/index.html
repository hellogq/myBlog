vueJS入坑-环境配置

2017-03-15

最近自己入了好多的坑，一入前端深似海，感觉自己已经跳不出来了==。这几天在看vue的东西，初识vue，看着好简单，结果光配置环境就配置了一下午（好想哭）各种报错，我最终归结于node和npm的版本太低，记得要升级哟！好了，我们回归正题，进行VueJS环境的配置。

node.js(必须安装)

基于node.js安装vue

  • 直接在cmd里面输入 npm install -g vue 全局安装vue （如果网络不是很好，建议使用淘宝镜像）

    安装全局vue-cli脚手架,用于帮助搭建所需要的框架

  • 在cmd里
    • 输入npm install vue-cli -g 回车等待安装
    • 输入 vue -V 查看版本号 证明安装成功

创建项目

在cmd里输入 vue init webpack vue_name(项目文件夹名称)
—–值得一提的是，直接运行这个命令会将文件创建在你安装的vue的文件目录下，可以cd到你想要放的文件夹的位置


等待一会，会出现

这样已经离成功不远了

安装依赖

  • 在cmd中

    • 输入： cd vue_name(创建的项目文件夹名)
    • 输入： npm install 回车 你会发现项目文件夹中生成了好多文件。node_modules文件夹里的文件就是之前安装的依赖


ok,离成功只有一步之遥了
7.测试环境是否搭建成功
在cmd里输入 npm run dev
运行起来如果出现一个页面

ok,完成啦！恭喜你 你已经进坑了【坏笑】
